# Nios V/M LED Control Using Avalon PIO

## Description

This project demonstrates LED control using the Avalon PIO IP core in a Nios V/M-based embedded system.

## Features

* LED control using memory-mapped I/O
* Console output using JTAG UART
* Simple embedded C implementation

## Hardware Used

* Cyclone V DE1-SoC Board

## Software Tools

* Quartus Prime
* Platform Designer
* RISCFree IDE

## Output

* LEDs turn ON based on the given pattern
* Console displays LED value
